## 🌼 What this project does

This project shows how K-Means clustering works on the Iris dataset. It uses a Jupyter Notebook so you can follow each step with clear examples and charts.

You will see how the model groups flowers based on their features. You will also learn how to pick a good number of clusters with the Elbow Method and how to check the result with the Silhouette Score.

## 🧪 How the notebook works

The notebook follows a simple flow:

1. Load the Iris dataset
2. Check the data shape and sample rows
3. Pick the feature columns
4. Fit the K-Means model
5. Create cluster labels
6. Draw the results
7. Use the Elbow Method to test cluster counts
8. Use the Silhouette Score to compare cluster fit

Each step builds on the one before it, so you can learn as you go.
